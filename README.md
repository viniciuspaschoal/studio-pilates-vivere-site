# Studio Pilates Vivere

Site institucional do Studio Pilates Vivere, criado com foco em uma apresentação leve, acolhedora e responsiva.

## Tecnologias

- React
- TanStack Router / Start
- Vite
- TypeScript
- Tailwind CSS
- Lucide React

## Como instalar

```bash
npm install
```

## Como rodar localmente

```bash
npm run dev
```

Por padrão, o projeto usa a porta `3000`.

## Como gerar build

```bash
npm run build
```

## Scripts úteis

```bash
npm run lint
npm run typecheck
npm run format:check
```

## Dados do cliente

Os textos, links públicos, WhatsApp, Instagram, endereço, depoimentos e demais conteúdos editáveis ficam centralizados em:

```txt
src/config/site.ts
```

Os dados reais do cliente podem ser atualizados posteriormente nesse arquivo, sem alterar a estrutura visual do site.

## Deploy

O projeto está preparado para versionamento no GitHub e importação posterior na Vercel.
